# SITCC Tutorial 1: First Game 🎮

Welcome to the first tutorial in the **SITCC MakeCode Arcade Tutorial Series**!

In this tutorial, you will learn how to create your first MakeCode Arcade game.

## What You Will Learn

By the end of this tutorial, you will know how to:

- 🎮 Create and control a player sprite
- 🗺️ Create a game map
- 💎 Add collectibles
- 👾 Create enemies
- 🏆 Create win and lose conditions

## Start the Tutorial

Click the link below to open the interactive MakeCode Arcade tutorial:

## Requirements

You need:
- A free Microsoft MakeCode Arcade account (optional)
- A web browser
- Basic computer skills

No previous programming experience is required!

## About SITCC

SITCC (St. Ignatius Tigers Coding Club) is a student-led coding club where members learn programming, game development, and computer science through hands-on projects.
